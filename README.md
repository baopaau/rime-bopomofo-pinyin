# 拼音輸入法轉寫注音符號

配方： ℞ **`bopomofo_pinyin`**

[Rime](https://rime.im) 注音輸入方案

使用方式同一般漢語拼音方案，

上屏碼和候選項註釋會轉寫成注音符號。

數字鍵12345輸入聲調。

Enter輸出原碼，

Ctrl-Enter輸出轉寫後的注音上屏碼。

映射規則抄自[注音](https://github.com/rime/rime-bopomofo) ℞ **`bopomofo`**。

本方案依賴於`terra_pinyin.dict.yaml` （[地球拼音](https://github.com/rime/rime-terra-pinyin) ℞ **`terra-pinyin`**）
